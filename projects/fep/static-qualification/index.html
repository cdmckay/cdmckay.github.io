In Fep classes, properties marked as static or const and methods marked as static are qualified using the :: character.

public static class Math
{
    const var Pi = 3.14159;
    const var E = 2.71;

    static def sin( x )
    {
        ...
    }
}

var radius = 10;
var circumference = 2 * Constants::Pi * radius;
var val = Math::sin 10;
